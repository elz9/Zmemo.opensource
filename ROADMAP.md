# Roadmap

This roadmap reflects current priorities and may change based on user feedback and technical constraints. Use GitHub Issues to propose or discuss items.

## Near Term

- Improve onboarding and credential setup flow
- Expose auto-sync controls more clearly in the UI
- Add richer attachment metadata display and filtering
- Expand stats panel with sync health and cache metrics

## Mid Term

- Saved searches and quick filters
- Multi-account support
- Export and backup tools for local data
- Smarter decision detection rules and explainability

## Longer Term

- Advanced semantic search options
- Keyboard-first navigation and shortcuts
- Label-aware filtering and views
- Cross-platform installer polish and update channels
